# Architecture Diagrams
This folder contains all VMEA system diagrams as standalone files. Each diagram includes a short summary to help readers quickly understand what they are looking at.

## Color Legend

| Color | Meaning |
|-------|--------|
| 🟦 Blue (`#3b82f6`) | Input / Source data |
| 🟨 Amber (`#f59e0b`) | Whisper transcription |
| 🟪 Purple (`#8b5cf6`) | LLM / Ollama processing |
| 🟩 Green (`#10b981`) | Output / Write operations |
| ⬜ Slate (`#64748b`) | State / Storage |
